# Background Music

Place licensed audio files here. The site currently looks for:

- `wavin-flag.mp3` for `K'NAAN - Wavin' Flag`
- `world-cup-track-2.mp3` for `Waka Waka`
- `world-cup-track-3.mp3` for `Ale Ale`
- `world-cup-track-4.mp3` for `Ishow Speed`
- `world-cup-track-5.mp3` for `Dreamers`
- `world-cup-track-6.mp3` for `Russia 2018`

To add more tracks, place the files in this folder and add them to `TRACKS` in
`app/components/layout/ExperienceControls.tsx`.

Do not commit copyrighted tracks unless you have the rights to distribute them with this website.
